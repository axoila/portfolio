Neon Overdrive (Code Sample)

Role: Programming, Design

This game was originally developed live at the Gamescom 2017 HSMW booth. We chose to make a competetive multiplayer game with simple controls, because the unique environment gave us new players every few minutes, with wildly variying skill levels, often in small groups.

I added a few quality of life changes like a pause menu as well improving the overall architecture and code quality after the event.

You can find the game at https://redye.itch.io/neon-overdrive
and the source code at https://github.com/axoila/neon-overdrive

No Time For Apocalypse

Role: Management, Design, Programming

A university project I did with two fellow students. In the game you help people with their tasks and then fight the horsemen of the apocalypse.

The game includes 4 levels with one bossfight each and a completely interactive dialogue System based on Yarnspinner.

Though we weren't able to get the game to a state where I'm 100% happy with it, we learned a lot from it as it and decided to release it as it is.
The core development time of the project was about 4 months with a bit polishing done by myself afterwards.

It can be found at https://redye.itch.io/no-time-for-apocalypse.
