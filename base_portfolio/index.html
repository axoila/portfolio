Role: Programming, Design

No Time For Apocalypse

Role: Management, Design, Programming

A university project I did with two fellow students. In the game you help people with their tasks and then fight the horsemen of the apocalypse.

The game includes 4 levels with one bossfight each and a completely interactive dialogue System based on Yarnspinner.

Though we weren't able to get the game to a state where I'm 100% happy with it we released it as it is because of time constraints. The core development time of the project was about 4 months with some polishing done by myself afterwards.

It can be found here.
